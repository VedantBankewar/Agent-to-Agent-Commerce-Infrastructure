# AgentTrade v2 — Autonomous Procurement Agent: Complete Implementation Plan

**Branch**: `autonomous-agent` (created from `main`)  
**Date**: 2026-05-19

---

## 1. Why This Rethink

AgentTrade v1 is a scripted pipeline — `demo.py` calls tools in fixed order, supplier quotes are generated by direct function calls (no LLM), and the 2-round negotiation tool is never invoked. The LangChain agents exist but are unused.

**v2 Goal**: A truly autonomous LLM-powered buyer agent that drives ALL decisions. The buyer fills a structured form (USD-denominated), clicks "Deploy", and the agent autonomously discovers suppliers, negotiates deeply (5-7 rounds, multi-variable), and locks an Algorand escrow — all while the buyer watches live.

---

## 2. Key Design Decisions

### 2.1 USD-First Pricing
- **All UI, agent communication, and database values in USD**
- Buyer sets budget in USD, sees prices in USD, negotiation happens in USD
- **Conversion to ALGO only at escrow lock time** — the `lock_escrow` tool converts USD → ALGO using a rate (live API or configurable demo rate)
- The escrow contract still operates in microALGO internally
- The buyer never sees ALGO unless they click the Algorand explorer link
- Store `usd_to_algo_rate` in the deal record for auditability

### 2.2 Structured Form Input (No Free Text Parsing)
Replace the single text input with structured fields. The `parse_procurement_goal()` regex parser is eliminated entirely.

**Required Fields:**
| Field | UI Type | Backend Key | Validation |
|---|---|---|---|
| Product Name | Text input | `item` | Non-empty, max 200 chars |
| Category | Dropdown | `category` | One of: furniture, office_supplies, electronics, general |
| Quantity | Number input | `quantity` | Integer > 0 |
| Max Budget (USD) | Currency input | `budget_usd` | Float > 0 |
| Delivery Deadline | Date picker | `deadline` | Future date, ISO 8601 |

**Optional Fields (collapsible "Advanced Options" section):**
| Field | UI Type | Backend Key | Default |
|---|---|---|---|
| Target Price/Unit (USD) | Currency input | `target_price_usd` | `null` (agent discovers market rate) |
| Minimum Warranty | Dropdown (1yr/2yr/3yr/5yr) | `min_warranty_yrs` | `1` |
| Priority | Radio: Cost / Speed / Quality | `priority` | `"balanced"` |
| Special Requirements | Textarea | `requirements` | `""` |

**Priority affects scoring weights:**
| Priority | Price Weight | Delivery Weight | Rating Weight | Warranty Weight |
|---|---|---|---|---|
| `cost` | 0.60 | 0.15 | 0.10 | 0.15 |
| `speed` | 0.20 | 0.50 | 0.15 | 0.15 |
| `quality` | 0.15 | 0.15 | 0.35 | 0.35 |
| `balanced` (default) | 0.40 | 0.30 | 0.20 | 0.10 |

**Preset Buttons**: Quick-fill all fields (not text strings). Example presets:
- "50 Office Chairs" → item: Ergonomic Office Chair, category: furniture, qty: 50, budget: $15,000, deadline: +30d, target: $250/unit, priority: balanced
- "100 Ballpoint Pens" → item: Ballpoint Pen, category: office_supplies, qty: 100, budget: $500, deadline: +14d, priority: cost
- "10 Standing Desks" → item: Standing Desk, category: furniture, qty: 10, budget: $8,000, deadline: +45d, target: $700/unit, priority: quality

### 2.3 Supplier-Agnostic Interface
The buyer agent communicates through a uniform `SupplierInterface` ABC. The buyer never knows if the other side is a bot, an LLM, or a human.

### 2.4 Deep Multi-Variable Negotiation
- Up to 7 rounds per supplier (configurable)
- Variables: price, delivery days, warranty years, quantity adjustments
- Concurrent negotiation with 3+ suppliers
- Agent uses strategy based on buyer's priority setting
- Agent can walk away, play suppliers against each other, trade variables

### 2.5 Frontend Decoupled via EventBus
- `core/events.py` provides in-process event emitter
- Agent emits events at key points (discovery, quote, counter, accept, escrow)
- `server.py` subscribes and translates to SSE for frontend
- CLI mode: no subscribers, events are no-ops
- Agent code has zero imports from `server.py` or `frontend/`

#### `core/supplier_interface.py`
```python
class SupplierInterface(ABC):
    supplier_id: str
    supplier_name: str

    @abstractmethod
    def receive_rfq(self, message: NegotiationMessage) -> NegotiationMessage: ...

    @abstractmethod
    def receive_counter(self, message: NegotiationMessage) -> NegotiationMessage: ...

    @abstractmethod
    def confirm_deal(self, message: NegotiationMessage) -> NegotiationMessage: ...

class SupplierRegistry:
    """Maps supplier_id -> SupplierInterface. Falls back to RuleBotSupplier."""
    _suppliers: dict[str, SupplierInterface] = {}

    @classmethod
    def register(cls, supplier_id: str, impl: SupplierInterface): ...
    @classmethod
    def get_or_default(cls, supplier_id: str) -> SupplierInterface: ...
```

```python
class NegotiationSessionManager:
    """Central orchestrator for all concurrent negotiations in a procurement."""
    
    def __init__(self, context: ProcurementContext, db_path: str): ...

    def start_negotiation(self, supplier_id: str, rfq_msg: NegotiationMessage) -> NegotiationMessage:
        """Send RFQ to supplier via SupplierRegistry. Phase: INVITED → QUOTED."""

    def send_counter(self, supplier_id: str, terms: NegotiationTerms, reasoning: str) -> NegotiationMessage:
        """Send counter-offer. Phase: QUOTED/NEGOTIATING → NEGOTIATING(round+1) or ACCEPTED/REJECTED."""

    def accept_offer(self, supplier_id: str) -> NegotiationMessage:
        """Accept current terms. Phase → ACCEPTED."""

    def reject_supplier(self, supplier_id: str, reason: str) -> None:
        """Walk away. Phase → REJECTED."""

    def get_status(self) -> dict:
        """Current state of all negotiations."""

    def get_best_offers(self) -> list[tuple[str, NegotiationTerms, float]]:
        """Active negotiations sorted by score."""

    def _score_terms(self, terms: NegotiationTerms, supplier_rating: float) -> float:
        """Score using priority-weighted formula."""

    def _persist_round(self, supplier_id: str, msg: NegotiationMessage) -> None:
        """Write to negotiation_rounds table in SQLite."""
```

### Layer 3: `agents/buyer/` — Autonomous Buyer Agent

#### `agents/buyer/tools.py` — 8 Buyer Tools

```python
def build_buyer_tools(session_manager: NegotiationSessionManager) -> list[BaseTool]:
    """Build all 8 buyer tools, each wrapping the session manager."""

# Tool 1: discover_suppliers(category: str) -> dict
#   Wraps tools/search_suppliers.py
#   Returns {count, suppliers: [{id, name, rating, base_cost, lead_days, warranty_yrs}]}

# Tool 2: request_quotes(supplier_ids: list[str]) -> dict
#   For each supplier: session_manager.start_negotiation()
#   Concurrent via ThreadPoolExecutor
#   Returns {supplier_id: {terms, score}} for each responding supplier

# Tool 3: evaluate_quotes() -> dict
#   Calls session_manager.get_best_offers()
#   Returns ranked list with scores, using priority-weighted formula
#   Includes budget check (flag any over-budget suppliers)

# Tool 4: send_counter_offer(supplier_id: str, unit_price_usd: float,
#          delivery_days: int, warranty_yrs: float, message: str) -> dict
#   Calls session_manager.send_counter()
#   Returns {decision, response_terms, round_number, supplier_message}

# Tool 5: accept_offer(supplier_id: str) -> dict
#   Calls session_manager.accept_offer()
#   Returns {confirmed, final_terms}

# Tool 6: reject_supplier(supplier_id: str, reason: str) -> dict
#   Calls session_manager.reject_supplier()

# Tool 7: lock_escrow(supplier_id: str) -> dict
#   1. Get agreed terms from session_manager
#   2. Fetch USD→ALGO rate
#   3. Convert amount: amount_algo = amount_usd / rate
#   4. Call tools/sign_escrow.py (with ALGO amount)
#   5. Record usd_to_algo_rate in deal
#   Returns {deal_id, txid, confirmed_round, amount_usd, amount_algo, deal_hash}

# Tool 8: get_negotiation_status() -> dict
#   Returns {deal_phase, negotiations: {supplier_id: {phase, round, current_terms, score}}}
```

```
4. Negotiate with top 2-3 suppliers concurrently for competitive pressure
5. Trade variables: if price is stuck, push for better delivery or warranty
6. Use target price as anchor if provided, but don't reveal it to suppliers
7. Walk away if: 5+ rounds with no progress, or all offers exceed budget
8. Accept when: within budget AND meets deadline AND score > 70

DECISION FRAMEWORK:
  After initial quotes:
    - If any quote is at or below target price → accept immediately
    - If all quotes are over budget → negotiate hardest with cheapest supplier
    - If spread is tight (<5%) → focus on non-price variables
  During negotiation:
    - Reference competing offers without lying ("I have a competitive offer at...")
    - Counter with terms that split the difference, weighted toward priority
    - On {priority} priority: {specific guidance per priority}
  Final decision:
    - Compare all ACCEPTED/latest offers using scoring formula
    - Pick highest score that's within budget and meets deadline
    - If no offers meet constraints → report failure to buyer

CONSTRAINTS:
  - Never expose private keys, mnemonics, or API keys
  - All prices in USD (conversion to ALGO happens at escrow lock)
  - Always verify escrow transaction confirmed on-chain
  - Maximum 30 tool calls total (prevents infinite loops)
```

## 7. Server Changes (`server.py`)

### EventBus-Based SSE
Replace subprocess stdout streaming with EventBus subscription:

```python
@app.post("/api/run_pipeline")
async def run_pipeline(request: ProcurementRequest):
    """Run the autonomous buyer agent, stream events via SSE."""
    
    async def event_stream():
        queue = asyncio.Queue()
        
        def on_event(event_type, data):
            queue.put_nowait((event_type, data))
        
        EventBus.subscribe(on_event)
        
        # Run agent in background thread
        loop = asyncio.get_event_loop()
        agent_task = loop.run_in_executor(None, run_buyer_agent, request)
        
        try:
            while not agent_task.done():
                event_type, data = await asyncio.wait_for(queue.get(), timeout=1.0)
                yield f"data: {json.dumps({'type': event_type, **data})}\n\n"
        except asyncio.TimeoutError:
            pass
        
        yield "data: [DONE]\n\n"
        EventBus.clear()
    
    return StreamingResponse(event_stream(), media_type="text/event-stream")
```
